# Site Web - Antenne Jeunes Gabrielle Dinard ## 📋 Description du projet Ce projet est un site web WordPress pour l'**Antenne Jeunes de Gabrielle Dinard**. Il utilise le thème Divi avec un thème enfant personnalisé pour gérer les événements et les actualités de l'association. ## 🏗️ Structure du projet ``` antenne_jeunes_gabrielle_dinard/ ├── wp-content/ │ └── themes/ │ ├── Divi/ # Thème parent Divi │ └── divi-child/ # !!! THÈME ENFANT PERSONNALISÉ │ ├── functions.php # Fonctions personnalisées │ ├── style.css # Styles CSS personnalisés │ ├── front-page.php # Template page d'accueil │ └── README.md # Documentation détaillée ├── wp-config.php # Configuration WordPress └── index.php # Point d'entrée WordPress ``` ## 🎯 Accès au code personnalisé **Pour voir le code personnalisé, allez dans :** ``` wp-content/themes/divi-child/ ``` ## 📁 Que contient le thème enfant ? ### 🔧 Fonctionnalités principales 1. **Type de contenu personnalisé "Événements"** - Gestion des événements de l'antenne - Champ date personnalisé pour chaque événement - Interface d'administration WordPress 2. **Page d'accueil personnalisée** - Affichage automatique des 6 derniers événements - Design moderne avec dégradés et effets hover - Responsive et optimisé pour tous les écrans 3. **Styles CSS personnalisés** - Import du style Divi parent - Design moderne avec animations - Couleurs et typographie adaptées à l'identité visuelle ### 📝 Fichiers importants - **`functions.php`** : Contient toutes les fonctions personnalisées - **`style.css`** : Styles CSS pour l'affichage des événements - **`front-page.php`** : Template de la page d'accueil - **`README.md`** : Documentation détaillée du thème enfant ## 🚀 Installation et utilisation ### Prérequis - WordPress installé - Thème Divi activé - Accès à l'administration WordPress ### Installation 1. Le thème enfant est déjà installé dans `wp-content/themes/divi-child/` 2. Activer le thème "Divi Child" dans **Apparence > Thèmes** 3. Le site est prêt à utiliser ! ### Gestion des événements 1. Aller dans **Événements** dans le menu d'administration 2. Cliquer sur **Ajouter un événement** 3. Remplir le titre, le contenu et la date de l'événement 4. Publier l'événement 5. Il apparaîtra automatiquement sur la page d'accueil ## 🛠️ Technologies utilisées - **WordPress** : CMS de base - **Divi** : Thème parent d'Elegant Themes - **PHP** : Langage de programmation - **CSS3** : Styles avec dégradés et animations - **HTML5** : Structure sémantique ## 📖 Documentation détaillée Pour plus d'informations sur le thème enfant et ses fonctionnalités, consultez : ``` wp-content/themes/divi-child/README.md ```
